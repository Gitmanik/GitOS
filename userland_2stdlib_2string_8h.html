Parameters
ptr1Pointer to block of memory
ptr2Pointer to block of memory
lenNumber of bytes to compare
Returns
int
  • 0 if matches,

  • 1 the first byte that does not match in both memory blocks has a greater value in ptr1 than in ptr2
